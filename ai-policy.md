# 🤖 AI Usage Policy

You are allowed to use AI tools (ChatGPT, Copilot, etc.) during this challenge.

However, transparency is required.

You must include a file named:

```
ai.md
```

This file must describe:

- Where AI was used
- What type of assistance it provided
- What parts were fully written by you
- How you validated or reviewed AI-generated code

We are not evaluating whether you use AI.
We are evaluating how responsibly and effectively you use it.

Failure to include `ai.md` may result in disqualification.
